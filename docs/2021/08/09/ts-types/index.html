映射类型

映射类型建立在索引签名的语法之上，用于声明尚未提前声明的属性类型：

映射类型是一种泛型类型，它使用PropertyKeys的联合（通常通过 a keyof创建）来迭代键以创建类型：

type OptionsFlags = {
[Property in keyof Type]: boolean;
};

映射修饰符
有两个额外的修饰符可以在映射期间应用：readonly和?分别影响可变性和可选性。

您可以通过添加-或前缀来删除或添加这些修饰符+。如果不添加前缀，则+为默认值

键重映射
